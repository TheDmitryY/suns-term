# Suns-term

Suns-term - my own python script for .profile in termux.
## Use Reference

Change ip and port of your host in menu file
```
   cd /settings 
   nvim menu.py
```
or 
```
    cd /settings
    nano menu.py
```

Start

```
python3 main.py
```



## Installation

Install my-project with git.

```bash
  git clone https://github.com/TheDmitryY/suns-term
  python3 main.py
```
